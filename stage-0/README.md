# HNG 14 Internship — Stage 0

## 📌 Task

Build a **testable, accessible, and responsive Todo Task Card** using HTML, CSS, and JavaScript.

---

## 🚀 Live Demo

https://augustine-hng-14-stage-0.vercel.app

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript

---

## ✅ Features

- Accessible and semantic HTML structure
- Fully keyboard navigable
- Dynamic time remaining calculation
- Responsive layout (320px → desktop)
- Task completion toggle (updates UI state)
- Priority and status indicators
- Tag system with testable identifiers

---

## 🧪 Testability

All required `data-testid` attributes are implemented for automated testing.
